#  Previsão de Churn em Telecomunicações

Este projeto foi desenvolvido para identificar padrões de cancelamento de clientes (Churn) e fornecer uma ferramenta preditiva que auxilie na retenção proativa. Utilizei técnicas de Machine Learning para transformar dados brutos em insights estratégicos.

###  O Problema de Negócio
Reter um cliente é mais estratégico e econômico do que adquirir um novo. O objetivo aqui foi entender quais comportamentos sinalizam que um cliente está prestes a deixar a empresa, permitindo uma intervenção antes do cancelamento.

###  Tecnologias e Ferramentas
- **Linguagem:** Python 3.13
- **Bibliotecas:** Pandas, NumPy, Scikit-Learn e Seaborn/Matplotlib.
- **Ambiente:** VS Code com Jupyter Notebooks.

###  Descobertas Principais
- **Tipo de Contrato:** Clientes com contratos mensais apresentam uma taxa de churn muito maior que os fidelizados.
- **Forma de Pagamento:** O boleto bancário é um forte indicador de risco em comparação ao débito automático.
- **Engajamento Inicial:** Clientes nos primeiros 12 meses exigem atenção redobrada.

### Organização do Projeto
- `/data`: Arquivos de dados e output de clientes prioritários.
- `/notebooks`: Documentação do processo de análise e treino do modelo.
- `/reports`: Gráficos de performance e importância de variáveis.
